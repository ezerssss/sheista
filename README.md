# sheista

Personal [ThemeCP](https://codeforces.com/blog/entry/136704) inspired competitive programming training tracker for Codeforces.

🏷️ **Status:** Active
🤖 **Vibe Coded:** Yes

---

## What it does

- Automatic creation of a 4-problem round to be solved within 2 hours.
- Auto detection when a problem is solved (30s polling).
- Tracking of problem type (greedy, math, etc.) mastery.
- Calendar heatmaps and streak counters.
